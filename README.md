# Projetos de Teste - Python

Este repositório contém três programas em Python com arquivos JSON, criados como desafios de teste para vaga de desenvolvedor.  
Cada projeto é independente e está organizado em sua própria pasta.

---

## Estrutura do Repositório

- `Desafio 1/`  
  - **Arquivos:** `desafio1.py`, `vendas_desafio1.json`  
  - **Descrição:** Calcula o valor total de vendas a partir de um arquivo JSON.  
  - **Apenas rodar o código com o arquivo correspondente ou outro arquivo com dados similares, para que faça o calculo e gere o texto, como foi proposto no desafio.**  
    ```bash
    python desafio1.py
    ```

- `Desafio 2/`  
  - **Arquivos:** `desafio2.py`, `estoque_desafio2.json`, `movimentacoes.json`  
  - **Descrição:** Gerencia estoque e movimentações de produtos, calculando totais e atualizações.  
  - **O código pode ser rodado com um arquivo de estoque já pronto ou sem ele. caso sem, ele mesmo cria os arquivos necessários. Menu interativo via  terminal.**  
    ```bash
    python desafio2.py
    ```

- `Desafio 3/`  
  - **Arquivos:** `desafio3.py`  
  - **Descrição:** Calcula juros e multas com base em datas de vencimento e valores fornecidos.  
  - **Não exige nenhum arquivo esterno, apenas pede ao usuário uma data e valor para realizar um cálculo.**  
    ```bash
    python desafio3.py
    ```

---

## Requisitos

- **Python 3.x**  
- Nenhuma biblioteca externa necessária (padrão Python)  

---

- Pastas como `__pycache__/`, `venv/` e `.idea/` estão ignoradas no repositório.  
- Todos os arquivos `.py` e `.json` necessários estão incluídos.  
- Este repositório foi organizado como portfólio profissional para desafios de vaga de desenvolvedor.
